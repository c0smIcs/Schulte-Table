# 🧠 Schulte Table Trainer

Веб-приложение для тренировки периферического зрения, внимания и навыков скорочтения с использованием Таблиц Шульте.

Проект представляет собой классическую сетку 5x5, где пользователю необходимо как можно быстрее найти числа от 1 до 25 в правильном порядке. Приложение реализует полный цикл Full-stack разработки: от серверной генерации логики до реактивного интерфейса и сохранения результатов в базе данных.

## ✨ Основные возможности

*   **Игровая механика:** Генерация случайных таблиц Шульте 5x5 на сервере.
*   **Real-time Таймер:** Секундомер работает на сервере и транслируется клиенту через **Server-Sent Events (SSE)**, что исключает возможность читерства на клиенте.
*   **Личные рекорды:** Лучшее время сохраняется в **PostgreSQL**.
*   **Сессии:** Идентификация пользователей через Cookies (без регистрации).
*   **Адаптивный UI:** Интерфейс оптимизирован как для десктопов, так и для мобильных устройств.

## 🛠 Технологический стек

### Backend
*   **Language:** Go (Golang)
*   **Database:** PostgreSQL
*   **ORM:** GORM (для работы с БД и миграций)
*   **Architecture:** REST API + SSE (EventSource)
*   **Features:** Dependency Injection, Graceful handling, Goroutines for async tasks.

### Frontend
*   **HTML / CSS:** Flexbox верстка, CSS Variables.
*   **JavaScript:** Vanilla JS (ES6+), Fetch API, DOM Manipulation.
